The Challenge

Trust in an Autonomous World

As agents act independently, the need for verifiable trust becomes critical.

The Problem

How do you trust an agent you've never interacted with? Today there is no standardized way for agents to prove reliability, leaving humans and systems guessing.

The Solution

Promise-based trust — objective, portable, queryable. Agents make promises, submit evidence, and earn trust scores that anyone in the network can verify.

Developer Experience

Up and Running in Minutes

Install the Python SDK and start tracking trust in a few lines of code.

quickstart.py 
from soz_ledger import SozLedgerClient

client = SozLedgerClient(api_key="your_api_key")

# Register an agent
agent = client.entities.create(
    name="DataFetchBot",
    type="agent",
    metadata={"platform": "langchain", "version": "1.0"}
)

# Create a promise
promise = client.promises.create(
    promisor_id=agent.id,
    promisee_id="user_abc",
    description="Fetch and deliver report within 30 seconds",
    deadline="2026-02-10T15:00:00Z",
    category="delivery"
)

# Submit evidence and fulfill
client.evidence.submit(
    promise_id=promise.id,
    type="api_callback",
    submitted_by=agent.id,
    payload={"status": 200, "delivered_at": "2026-02-10T14:59:45Z"}
)
client.promises.fulfill(promise.id)

# Check trust score
score = client.scores.get(agent.id)
print(f"Trust Score: {score.overall_score}")  # 0.85
print(f"Level: {score.level}")                # "Highly Trusted"
